# Dizzi
## A simple discord bot made with discord.py.
### Features:
* Timezone Conversion
* Server-Wide and Personal Emoji Tracking
* Slash and Prefix Command Support
* JoJo's Bizarre Adventure Stand Creation Mini-Game utilizing Stable Diffusion
* Birthday Reminders
* And much more!
